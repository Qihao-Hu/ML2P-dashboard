# ML2P dashboard snapshot

This repository contains only the static files required to display a public
benchmark dashboard:

- `index.html`
- `assets/styles.css`
- `assets/dashboard.js`
- `dashboard-data.json`

It intentionally excludes the ML2P implementation, private repository history,
automation, raw measurements, credentials, and development configuration.

The bundled JSON contains a minimized snapshot of aggregate benchmark metrics.
Serve the repository root with GitHub Pages or any static HTTP server.
